# Frontend Development Skill

## Instructions

1. **Page Structure**
   - Use semantic HTML (header, main, section, footer)
   - Organize content with clear hierarchy
   - Follow accessibility standards (ARIA, alt text)

2. **Components**
   - Build reusable UI components (buttons, cards, modals, navbars)
   - Use props for customization
   - Keep components modular and clean

3. **Layout System**
   - Use Flexbox and CSS Grid
   - Ensure responsive design (mobile-first)
   - Maintain consistent spacing and alignment

4. **Styling**
   - Use modern CSS or Tailwind
   - Apply consistent color schemes
   - Follow design system rules (fonts, spacing, borders)

5. **Responsiveness**
   - Breakpoints for mobile, tablet, desktop
   - Fluid typography and images
   - Test across screen sizes

## Best Practices
- Keep components small and reusable  
- Use clear class naming  
- Avoid inline styles  
- Follow design consistency  
- Optimize for performance  

## Example Structure

```html
<main class="container">
  <header class="navbar">
    <h1 class="logo">My App</h1>
  </header>

  <section class="content">
    <div class="card">
      <h2>Title</h2>
      <p>Description text here.</p>
      <button class="btn-primary">Click Me</button>
    </div>
  </section>
</main>
